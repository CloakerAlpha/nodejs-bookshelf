# nodejs-bookshelf

Library yang digunakan adalah :
-ESLint untuk formatting penulisan kode
-HAPI untuk konfigurasi web server node.js

Penggunaan :
.Jalankan web server node.js
.Menggunakan postman collection untuk uji coba data yang diterima dan dikirim oleh apliaksi web server, apakah sesuai atau tidak.
